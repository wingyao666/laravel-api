## 服务启动
php artisan serve

指定域名
php artisan serve --host=api.lv7.com


## 创建接口模型类
php artisan make:controller TaskController //创建了TaskController的控制器

php artisan make:controller UserController --api //创建了UserController的控制器 --针对restful api

##查看路由布局
php artisan route:list
